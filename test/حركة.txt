## 🎯 مثال كامل شغال — 5 sections بأنيميشن مختلف لكل وحدة

```html
<section class="reveal" data-animate="animate-fade-up animate-duration-700">
  <h2 class="text-3xl font-bold">قسم 1</h2>
</section>

<section class="reveal" data-animate="animate-fade-left animate-duration-700">
  <h2 class="text-3xl font-bold">قسم 2</h2>
</section>

<section class="reveal" data-animate="animate-fade-right animate-duration-700">
  <h2 class="text-3xl font-bold">قسم 3</h2>
</section>

<section class="reveal" data-animate="animate-jump-in animate-duration-700">
  <h2 class="text-3xl font-bold">قسم 4</h2>
</section>

<section class="reveal" data-animate="animate-flip-up animate-duration-700">
  <h2 class="text-3xl font-bold">قسم 5</h2>
</section>

<section class="reveal" data-animate="animate-wiggle animate-duration-700">
  <h2 class="text-3xl font-bold">قسم 6</h2>
</section>
```

**CSS:**
```css
@import "tailwindcss";
@import "tailwindcss-animated";

.reveal {
  opacity: 0;
}
```

**JavaScript (نفس الكود اللي عندك، جاهز):**
```html
<script>
  const items = document.querySelectorAll('.reveal');

  const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        const classes = entry.target.dataset.animate.split(' ');
        entry.target.classList.remove('opacity-0');
        entry.target.classList.add(...classes);
        observer.unobserve(entry.target);
      }
    });
  }, {
    threshold: 0.2,
    rootMargin: '0px 0px -50px 0px'
  });

  items.forEach(item => {
    item.classList.add('opacity-0');
    observer.observe(item);
  });
</script>
```

